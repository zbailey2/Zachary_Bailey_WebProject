HTML Formatting

HTML offers many different elements that allow you to format text or structure content on a webpage. Formatting is essential for creating a accessible and visually appealing website.

Key Elements

Here are some common HTML formatting elements:

  • <b>: Makes text bold.
  • <i>: Italicizes text.
  • <u>: Underlines text.
  • <strong>: Indicates importance (bold).
  • <em>: Emphasizes text (italic).
  • <small>: Displays smaller text.
  • <mark>: Highlights text.
  • <del>: Represents deleted text.
  • <ins>: Represents inserted text.

Example of HTML Formatting

Below is an example demonstrating various HTML formatting elements: 


<p>This is a <b>bold</b> statement.</p>
<p>This is an <i>italic</i> statement.</p>
<p>This is a <u>underlined</u> statement.</p>
<p>This is a <strong>strongly emphasized</strong> statement.</p>
<p>This is an <em>emphasized</em> statement.</p>
<p>This is some <small>small text</small> and normal text.</p>
<p>This text is <mark>highlighted</mark> for emphasis.</p>
<p>This text was <del>deleted</del> and this text is <ins>inserted</ins>.</p>

This example shows different formatting elements in HTML to have better presentation of text on a webpage.
