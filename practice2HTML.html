Question 1: Which tag is used to make text bold?

  • <strong>
  • <b>
  • <bold>
  • Both <strong> and <b>

Question 2: What does the <em> tag do?

  • It emphasizes text.
  • It makes text larger.
  • It changes the color of text.
  • It underlines text.

Question 3: Which of the following tags is used for italic text?

  • <i>
  • <italic>
  • <em>
  • Both <i> and <em>
