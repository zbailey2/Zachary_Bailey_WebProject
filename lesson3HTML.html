HTML Visual Elements

HTML provides multiple elements that allow visual content in web pages. These elements increase the user experience by adding images, videos, and other multimedia elements.

Key Visual Elements

Here are some of the primary visual elements you will use in HTML:

  • <img>: Used to embed images in a webpage.
  • <video>: Used to embed videos in HTML.
  • <audio>: Used to add sound files to the webpage.
  • <iframe>: Used to embed another HTML page within the current page.
  • <figure> and <figcaption>: Used to enclose images and their captions.

Example of HTML Visual Elements

Below is an example demonstrating various HTML visual elements: 


<figure>
    <img src="example.jpg" alt="Example Image" width="300">
    <figcaption>This is an example image with a caption.</figcaption>
</figure>

<video controls>
    <source src="example.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

<audio controls>
    <source src="example.mp3" type="audio/mpeg">
    Your browser does not support the audio tag.
</audio>

<iframe src="https://www.example.com" width="600" height="400"></iframe>

This example shows how to use visual elements in your HTML files.
